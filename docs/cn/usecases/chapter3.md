# 第3章：龙虾（OpenClaw）实战案例

## 1. 案例来源

- 官方 Showcase：<https://docs.openclaw.ai/start/showcase>
- 社区用例：<https://github.com/hesamsheikh/awesome-openclaw-usecases>

## 2. 典型场景（精选）

1. PR 自动评审并回传 Telegram
2. 本地库存/酒窖 CSV 智能管理
3. 电商下单自动化（Browser）
4. Slack 自动客服 + 告警转发
5. 电话语音助手（Voice Bridge）
6. 多源技术资讯日报
7. 会议纪要自动分发到 Jira/Linear/Todoist
8. 家庭日历与家务协同
9. Second Brain 知识库
10. 本地 CRM 自动化

## 3. 落地模板

- 目标定义（成功标准）
- 输入源定义（数据来源）
- 动作链路定义（抓取→处理→输出）
- 安全边界定义（人工确认点）
- 指标定义（准确率/节省时间）

## 4. 安全提醒

!!! warning "第三方技能先审计"
    社区技能是“可用资源”，不是“默认可信”。
    涉及支付、删除、外发、系统控制的动作必须人工确认。
